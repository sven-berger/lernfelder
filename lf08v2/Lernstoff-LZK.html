L2: Python Konstruktor

Wählen Sie die richtigen Regeln aus, die für einen Python Konstruktor gelten.

  • Ein Konstruktor ist die Methode einer Klasse, die automatisch beim Erzeugen eines Objektes aufgerufen wird.

  • Der Name des Konstruktors ist in Python immer __init__ und in der Klammer folgt eine Liste mit Übergabeparametern.

  • Die Parameterliste muss self als ersten Übergabeparameter enthalten.

  • Die Parameterliste des Konstruktor darf außer dem self  Parameter leer bleiben.

  • Jede Python-Klasse besitzt nur einen Konstruktor.

  • Bei einer leeren Parameterliste des Konstruktors kann das Klammmerpaar nach dem __init__ entfallen.
